# python-text-adventure-game
Console-based text adventure game built in Python using dictionaries, loops, and conditional logic, where players navigate rooms, collect items, and avoid a losing condition.

## Features
- Room navigation
- Item collection
- Inventory tracking
- Win and lose conditions

## Concepts Used
- Dictionaries
- Lists
- Loops
- Conditional logic
- User input

## How to Run

1. Download or clone the repository
2. Open a terminal in the project folder
3. Run:

python game.py

## Example Gameplay

Move commands: north, south, east, west

Collect items by typing:
get [item]

Example:

You are in the Kitchen  
Inventory: []

Available moves: south, east  
Item in room: knife
